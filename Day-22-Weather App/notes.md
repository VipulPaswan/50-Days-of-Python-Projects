# Day 22 – Weather App using Python 🌦️

## 📌 Project Overview

This project is a Weather Application built using Python that fetches real-time weather data using an API.

Users can enter a city name and get live weather details such as temperature, humidity, and weather conditions.

---

## 🚀 Features

- Fetch real-time weather data
- Display temperature, humidity, and conditions
- Simple and easy to use
- Works with any city name

---

## 🛠 Technologies Used

- Python 3
- requests library
- JSON
