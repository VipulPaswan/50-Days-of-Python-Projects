# Day 11 – Internet Speed Test using Python 🌐

## 📌 Project Overview

This project is a **GUI-based Internet Speed Test Application** built using Python.  
It allows users to measure their **download speed, upload speed, and ping** directly from a desktop interface.

The application combines Python GUI development with network testing tools to create a simple and useful utility.

---

## 🚀 Features

- Check **Download Speed**
- Check **Upload Speed**
- Display **Network Ping**
- Simple and clean **GUI interface**
- One-click speed test

---

## 🛠 Technologies Used

- Python 3
- Tkinter (GUI Development)
- speedtest-cli library
