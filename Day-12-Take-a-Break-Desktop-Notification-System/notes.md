# Day 12 – Take a Break Desktop Notification System ⏰💻

## 📌 Project Overview

This project is a **Desktop Notification System** built using Python that reminds users to take regular breaks while working on their computer.

The application runs in the background and sends periodic notifications to encourage healthy work habits and reduce screen fatigue.

This type of automation is useful for developers, students, and professionals who spend long hours working on their computers.

---

## 🚀 Features

- Desktop notifications reminding you to take a break
- Runs automatically at fixed time intervals
- Lightweight and simple to use
- Helps maintain healthy work habits

---

## 🛠 Technologies Used

- Python 3
- plyer (desktop notification library)
- time module
