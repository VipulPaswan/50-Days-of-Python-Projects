# Day 9 – Google Translator using Python 🌐

## 📌 Project Overview

This project is a **Language Translator Desktop Application** built using Python.  
It allows users to translate text between multiple languages using Google Translate.

The application provides a simple graphical interface where users can enter text, select source and destination languages, and instantly get translated results.

---

## 🚀 Features

- Translate text between **100+ languages**
- Simple and clean **GUI interface**
- **Swap language** option
- **Clear text** functionality
- Error handling for empty input

---

## 🛠 Technologies Used

- Python 3
- Tkinter (GUI development)
- googletrans library
- messagebox module
