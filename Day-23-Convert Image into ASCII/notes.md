# Day 23 – Convert Image into ASCII Art using Python 🎨

## 📌 Project Overview

This project converts an image into ASCII art using Python.

ASCII art is a technique where images are represented using characters, creating a text-based version of the image.

---

## 🚀 Features

- Convert image into ASCII text
- Simple and quick processing
- Creative representation of images
- Easy to use

---

## 🛠 Technologies Used

- Python 3
- pywhatkit
