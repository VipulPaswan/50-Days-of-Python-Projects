# Day 19 – Python Virtual Environment & Requirements.txt 🧩

## 📌 Project Overview

This project demonstrates how to create and manage Python virtual environments using the built-in `venv` module.

It also shows how to generate and use a `requirements.txt` file to manage project dependencies.

---

## 🚀 Features

- Create virtual environment
- Activate and deactivate environment
- Install project-specific libraries
- Generate requirements.txt file
- Recreate environment using requirements file

---

## 🛠 Technologies Used

- Python 3
- venv module
- pip

---
