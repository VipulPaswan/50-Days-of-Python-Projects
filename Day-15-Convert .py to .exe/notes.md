# Day 15 – Convert Python Script (.py) to Executable (.exe) ⚙️

## 📌 Project Overview

This project demonstrates how to convert a Python script into a standalone executable (.exe) file using PyInstaller.

This allows Python applications to run on systems without requiring Python to be installed.

---

## 🚀 Features

- Convert .py file to .exe
- Create standalone executable
- Bundle all dependencies
- Easy application distribution

---

## 🛠 Technologies Used

- Python 3
- PyInstaller

---
