# Day 8 – Text to Handwriting Converter using Python ✍️

## 📌 Project Overview
This project converts normal typed text into a handwriting-style image using Python.

The program uses the Pillow (PIL) library to write text on an image using a handwriting font, making it appear like a handwritten assignment.

This type of tool can be useful for:
- Creating handwritten assignments
- Generating handwritten notes
- Educational demonstrations

---

## 🚀 Features

- Converts typed text into handwritten style
- Custom handwriting font support
- Automatic text wrapping
- Saves output as an image file

---

## 🛠 Technologies Used

- Python 3
- Pillow (PIL)
- textwrap module
