# Day 20 – Screenshot Tool using Python 📸

## 📌 Project Overview

This project is a simple screenshot tool built using Python.  
It captures the current screen and saves it as an image file.

---

## 🚀 Features

- Capture full screen
- Save screenshot as image file
- Simple and fast execution
- Useful for documentation and debugging

---

## 🛠 Technologies Used

- Python 3
- PyAutoGUI
