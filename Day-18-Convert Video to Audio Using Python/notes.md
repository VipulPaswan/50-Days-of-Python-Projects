# Day 18 – Video to Audio Converter (MP4 to MP3) 🎬➡️🎧

## 📌 Project Overview

This project converts a video file (MP4) into an audio file (MP3) using Python.

It extracts the audio stream from the video and saves it as an MP3 file.

---

## 🚀 Features

- Convert MP4 video to MP3 audio
- Extract high-quality audio
- Simple and easy to use
- Useful for music, lectures, and podcasts

---

## 🛠 Technologies Used

- Python 3
- MoviePy
