# Day 25 – Password Manager Pro using Python 🔐

## 📌 Project Overview

This project is an advanced Password Manager built using Python with a modern GUI.

It allows users to securely store, generate, and retrieve passwords with additional features like clipboard copy and password visibility toggle.

---

## 🚀 Features

- Save credentials (website, username, password)
- Generate strong random passwords
- Copy password to clipboard
- Show / hide password
- Search stored credentials
- JSON-based local storage

---

## 🛠 Technologies Used

- Python 3
- Tkinter (GUI)
- JSON
- Pyperclip
