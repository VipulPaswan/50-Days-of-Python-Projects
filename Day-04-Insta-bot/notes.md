# Day 4 – Instagram Automation Bot using Python 🤖

## 📌 Project Overview
This project demonstrates basic social media automation using Python.  
The script uses the **Instabot** library to automate common Instagram actions such as logging in, following users, uploading photos, sending messages, and retrieving followers.

This project helped me understand how Python can interact with external platforms through automation libraries.

---

## 🚀 Features

- Automated Instagram login
- Follow a specific user
- Unfollow a user
- Upload photos with captions
- Send direct messages
- Retrieve followers list

---

## 🛠 Technologies Used

- Python 3
- Instabot Library
- Automation scripting
