# Day 17 – Screen Recorder using Python 🎥

## 📌 Project Overview

This project is a simple screen recorder built using Python.  
It captures the screen continuously and saves the recording as a video file.

---

## 🚀 Features

- Record screen in real-time
- Save output as video file
- Lightweight and simple implementation
- Useful for tutorials and demos

---

## 🛠 Technologies Used

- Python 3
- OpenCV
- PyAutoGUI
- NumPy
