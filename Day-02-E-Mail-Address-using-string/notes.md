# Day 2 – Email Validator in Python

## Project Description
This Python program validates an email address using string methods and logical conditions.

## Concepts Used
- String methods
- Conditional statements
- Loops
- Input handling

## Validation Rules
- Email must be at least 6 characters
- First character must be alphabet
- Only one '@' allowed
- '.' must be at -3 or -4 position
- No spaces allowed

## Example
Input: vipul123@gmail.com  
Output: Right Email
