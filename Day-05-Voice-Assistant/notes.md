# Day 5 – Voice Assistant using Python 🎙️🤖

## 📌 Project Overview
This project is a simple voice assistant built using Python.  
The assistant can listen to voice commands, process them, and respond using speech.

It can perform several tasks such as:
- Telling the current time
- Searching on Google
- Opening YouTube
- Searching Wikipedia
- Telling jokes
- Responding to basic questions

---

## 🚀 Features

- Voice activation command
- Speech recognition using microphone
- Text-to-speech responses
- Google search automation
- Wikipedia information retrieval
- Joke generation
- Browser automation

---

## 🛠 Technologies Used

- Python 3
- speech_recognition
- pyttsx3
- webbrowser
- datetime
- wikipedia
- pyjokes
