# Day 16 – Exploring App Development with Kivy & KivyMD 📱

## 📌 Project Overview

This project demonstrates basic app development using Python with Kivy and KivyMD.

Kivy is a framework used to build cross-platform applications, and KivyMD provides modern Material Design UI components.

---

## 🚀 Features

- Simple app layout
- Material Design UI using KivyMD
- Cross-platform app structure
- Interactive UI elements

---

## 🛠 Technologies Used

- Python 3
- Kivy
- KivyMD
