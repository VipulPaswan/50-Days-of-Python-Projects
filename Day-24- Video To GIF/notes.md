# Day 24 – Video to GIF Converter using Python 🎬➡️🎞️

## 📌 Project Overview

This project converts a video file into a GIF using Python.

It extracts frames from the video and combines them into a looping GIF animation.

---

## 🚀 Features

- Convert video to GIF
- Create animated GIFs
- Lightweight output format
- Easy to use

---

## 🛠 Technologies Used

- Python 3
- MoviePy
